Introduction

Welcome to the Animal Shelter Management API. This portfolio project demonstrates clean RESTful design, well‑structured OpenAPI documentation, and a developer‑friendly docs layout.

Authentication

All endpoints require an API key in the Authorization header:

Authorization: Bearer <your-api-key>

For demo purposes on GitHub Pages, you can use any placeholder key.

Animals Endpoints

  • GET /animals – Retrieve all animals
  • POST /animals – Create a new animal
  • GET /animals/{id} – Retrieve a specific animal
  • PUT /animals/{id} – Update animal data
  • DELETE /animals/{id} – Remove an animal

Success Stories Endpoints

  • GET /success – Retrieve adoption stories
  • POST /success – Create new story
  • GET /success/{id} – Retrieve a specific story
  • PUT /success/{id} – Update story
  • DELETE /success/{id} – Remove a story
